Setup & Instructions

  1. Create a Firebase project and enable Cloud Firestore.
  2. Add a web app and copy the firebaseConfig object.
  3. Paste your firebaseConfig into the FIREBASE_CONFIG placeholder in the HTML file.
  4. Open the file in a browser — the app will connect and sync realtime.

Note: For quick testing you can create the Firestore DB in test mode. Before sharing widely, apply stricter security rules — examples included in this Canvas.
